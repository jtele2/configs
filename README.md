# Joe Caprino's Configs

Install zsh and oh-my-zsh prior to using these configs. Symlink the following:  

- completions: `cp -as /home/ubuntu/configs/completions /home/ubuntu/.zsh`
- .vimrc: `ln -sf /home/ubuntu/configs/.vimrc /home/ubuntu/.vimrc`
- .zshenv: `ln -sf /home/ubuntu/configs/.zshenv /home/ubuntu/.zshenv`
- .zshrc: `ln -sf /home/ubuntu/configs/.zshrc /home/ubuntu/.zshrc`
- aliases: `ln -sf /home/ubuntu/configs/aliases.zsh /home/ubuntu/.oh-my-zsh/custom/aliases.zsh`
- functions: `ln -sf /home/ubuntu/configs/functions.zsh /home/ubuntu/.oh-my-zsh/custom/functions.zsh`
